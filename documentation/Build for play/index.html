🚀 Rocket Escape

A fast-paced arcade game where players control a rocket through space, avoid obstacles, and survive as long as possible. This project combines a Unity game with real-world hardware input using an Arduino-compatible controller, creating a physical and interactive gameplay experience.

Game Concept

Rocket Escape is a minimal arcade survival game built in Unity. The player controls a small rocket in deep space and must avoid incoming asteroids while staying inside the screen boundaries.

The main idea behind this project is to combine simple gameplay mechanics with a custom external controller, making the experience more tactile, physical, and engaging.

Background Story

You are the pilot of a small rocket lost in deep space.

Asteroids are constantly drifting toward you, and your only mission is to survive as long as possible.

With limited control, you must carefully navigate through the chaos and keep your rocket from crashing or drifting into the void.

Goal

  • Avoid all incoming obstacles
  • Stay within the screen boundaries
  • Survive as long as possible
  • Achieve the highest score

The score increases over time, rewarding players who can stay alive longer.

Inspiration

This project is inspired by classic arcade games such as:

  • Flappy Bird
  • Jetpack-style games
  • Endless runner mechanics

The goal was to create a simple but addictive gameplay loop while integrating physical interaction.

Game Mechanics

  • Press and hold to thrust upward
  • Release to fall downward
  • Obstacles move from right to left
  • Collision = Game Over
  • Flying off-screen (top or bottom) = Game Over

The rocket tilts upward while thrusting and tilts downward while falling, giving the player visual feedback and a stronger sense of motion.

Controls

Keyboard:

  • Press SPACE to control the rocket

External Controller (Arduino / ESP32):

  • A physical button is used as the main input
  • Press and hold the button to thrust upward
  • Release the button to fall

This creates a more immersive and tactile gameplay experience than using the keyboard alone.

External Hardware

This game integrates an Arduino-compatible ESP32 board with Unity through serial communication.

  • The button press is detected by the ESP32
  • The board sends "PRESS" and "RELEASE" signals through USB
  • Unity reads those signals and maps them to rocket movement

This creates a direct connection between physical interaction and on-screen action.

Player Feedback

  • "The physical button makes it feel much more engaging than just using a keyboard."
  • "Simple mechanics, but surprisingly challenging."
  • "The rocket movement feels smooth and responsive."
  • "It reminds me of Flappy Bird, but with a cooler space theme."

Some players found the game difficult at first, but improved quickly after a few rounds and began to enjoy the timing-based challenge.

Project Reflection

This project explores how a small number of mechanics can still create a compelling play experience. By combining Unity physics, simple obstacle avoidance, and a custom external controller, the game becomes more than just a screen-based interaction.

The use of an Arduino-compatible controller made the game feel more physical, playful, and performative during playtesting.
